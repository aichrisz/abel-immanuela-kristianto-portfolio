# Suraj Kale Portfolio

A strict dark single-page React + Tailwind portfolio implementation based on the provided design brief.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Phosphor Icons

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Notes

- Solid page background: `#050505`
- Fixed hero sits behind scrolling content
- Non-hero, non-footer sections use `max-w-[1440px]`
- Project cards are sticky stacked cards with `800px` fixed height
- Custom cursor dot uses `mix-blend-difference`
